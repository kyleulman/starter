# SvelteKit Starter

An opinionated SvelteKit template with TypeScript & Tailwind.

## What's included:

- TypeScript
- Prettier
- ESLint
- Tailwind CSS
- PWA support
- Share card placeholders
- A head component for managing page metadata ([@kyuisonline/workbench](https://www.npmjs.com/package/@kyuisonline/workbench))
